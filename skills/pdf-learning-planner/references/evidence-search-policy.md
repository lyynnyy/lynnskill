# Evidence Search Policy

Use this reference when a plan depends on current or evidence-based learning methods.

## When to Browse

Browse the web when the user asks for:

- current, latest, advanced, global, scientific, or evidence-based methods
- subject-specific pedagogy recommendations
- learning standards or benchmarks
- citations, links, or source attribution
- high-stakes educational decisions

Do not rely only on memory for these claims.

## Source Priority

Prefer sources in this order:

1. Official education or standards bodies, e.g. IES/WWC, EEF, NCTM, NCETM, Singapore MOE, national curriculum agencies.
2. Peer-reviewed reviews, meta-analyses, or university research centers.
3. Reputable education nonprofits or professional associations.
4. Publisher or product pages only for understanding the material, not for independent evidence claims.

For technical or research claims, use primary or authoritative sources. Cite sources in the final plan.

## Search Pattern

Run several narrow searches:

- `[subject] learning progression primary school evidence`
- `[subject] formative assessment young learners guidance`
- `[method] [subject] evidence review`
- `[country or body] [subject] curriculum progression`
- `site:ies.ed.gov [subject] practice guide`
- `site:educationendowmentfoundation.org.uk [subject] guidance report`

Record:

- source title and URL
- date or version when visible
- what principle it supports
- how it maps to the plan
- limits or age-range mismatch

## Evidence-to-Plan Translation

Translate evidence into small teaching moves:

- Spiral practice: revisit the same ability at increasing complexity.
- Retrieval practice: short recall or explanation prompts.
- Interleaving: mix related problem types after initial confidence.
- CPA or representation sequence: concrete, pictorial, abstract.
- Project-based learning: use a small real task after prerequisite skills.
- Formative assessment: define observable checks, not just scores.
- Metacognition: ask the learner to explain choices and verify reasonableness.

Do not list methods decoratively. Each method must affect sequencing, task design, checks, or review.
